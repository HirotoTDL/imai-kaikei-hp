## Gemini分析に基づく改善方針（2026/4/16策定）

### AI製感を消すポイント
1. テンプレ的な美辞麗句を排し、地域固有の表現を入れる
2. 「中津川」「恵那」「東濃」を具体的に散りばめる
3. 機能説明→ベネフィット訴求に書き換え
4. FAQ等の口調を親しみやすく
5. 写真プレースホルダーは後日実写真に差替え前提

### コンバージョン最適化
1. CTAの文言を具体化（「無料相談」→「初回無料・お気軽にご相談ください」）
2. CTAにマイクロコピー追加
3. 電話番号を目立たせる（シニア層は電話が主流）
4. フォーム項目を最小限に

### シニア層対応
1. 本文フォントサイズ16px以上
2. ボタン44px以上
3. 高コントラスト維持
4. 専門用語を避けた平易な表現

## セクション構成
1. Hero（キャッチコピー + CTA）
2. About（理念・3つの柱）
3. Services（6サービス・ベネフィット訴求）
4. CTA Banner（問い合わせ誘導）
5. Team（2名紹介 + 個性的プロフィール）
6. FAQ（5項目・親しみある口調）
7. Access（Google Maps + 事務所情報）
8. Contact（フォーム + 電話）
9. Footer
